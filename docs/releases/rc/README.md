# Release Candidates

Pre-production builds and sign-off documentation before promoting to production.

**Naming:** `rc-N-short-description.md`

Use [release-template.md](../templates/release-template.md) with **Release Candidate** set to `RC-N` and **Environment** set to Staging. Move finalized record to [history/](../history/) when promoted.

Update [history/index.md](../history/index.md) RC table during candidate testing.
