# Release Archive

Completed releases moved here after the 7-day review. Each archived release should be a folder or single markdown file.

**Naming:** `YYYY-MM-DD-vX.Y.Z/` or `YYYY-MM-DD-vX.Y.Z.md`

**Contents (recommended):**

- Release record (from [history/](../history/))
- Metrics snapshots (T+0 through T+7d)
- Incident records (if any)
- Rollback record (if applicable)

See [automation-architecture.md](../automation-architecture.md) for future `release:archive` automation.
