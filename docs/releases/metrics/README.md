# Release Metrics

Operational metrics snapshots captured at T+0, T+24h, T+48h, and T+7d after deploy.

**Naming:** `YYYY-MM-DD-vX.Y.Z-metrics.md` (append `-T24h` suffix if multiple files per release)

Copy [metrics-template.md](../templates/metrics-template.md) for each capture window. Summary rows belong in the parent release record **Operational Metrics** section.
