# Founder Launch Book™

Definitive operational handbook for BamSignal — written so a future CTO or operations lead can run the platform without tribal knowledge.

## Build exports

```bash
npm run build:founder-launch-book
```

Outputs (versioned + `latest`):

- `docs/founder-launch-book/exports/founder-launch-book-v{version}.md`
- `docs/founder-launch-book/exports/founder-launch-book-v{version}.pdf.html` (print to PDF)
- `docs/founder-launch-book/exports/latest.md`
- `docs/founder-launch-book/exports/latest.pdf.html`
- `docs/founder-launch-book/exports/manifest.json`

## Source chapters

Edit files in `docs/founder-launch-book/chapters/`, then rebuild.
