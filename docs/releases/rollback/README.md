# Rollback Records

Execution records for production rollbacks. One file per rollback event.

**Naming:** `YYYY-MM-DD-rollback-from-vX-to-vY.md`

Copy [rollback-template.md](../templates/rollback-template.md) when initiating rollback. Link from the affected release record in [history/](../history/).

Operational playbook: [deployment-recovery.md](../../runbooks/deployment-recovery.md)
