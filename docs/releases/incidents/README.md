# Release Incidents

Incident records tied to deployments and release windows.

**Naming:** `INC-YYYY-MM-DD-NNN.md`

Copy [incident-template.md](../templates/incident-template.md) when opening a release-related incident. Cross-link from the parent release record **Incident Log** section.

For general disaster recovery (non-release), see [runbooks](../../runbooks/README.md).
