# Hotfix Releases

Emergency production patches that ship outside the normal release cycle.

**Naming:** `YYYY-MM-DD-hotfix-short-description.md`

Use [release-template.md](../templates/release-template.md) with **Release Status** set to **Hotfix**. Complete [production-release-checklist.md](../checklists/production-release-checklist.md) with abbreviated scope where safe; never skip rollback plan or production verification.

Update [history/index.md](../history/index.md) hotfixes table.
